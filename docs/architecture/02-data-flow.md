## 二、Fallback 故障恢复机制

```text
┌──────────────────────────────────────────────────────────────────────────────────┐
│                         故障场景 · Fallback 恢复链路                              │
└──────────────────────────────────────────────────────────────────────────────────┘

 ┌───────────────────────────────────────────────┐
 │  场景1: 知识库检索无结果                        │
 │                                               │
 │  Qdrant.search_knowledge() ──→ 0条结果        │
 │        │                                       │
 │        ▼                                       │
 │  Fallback: AnySearch 联网搜索 ──→ 提取网页内容  │
 │        │                                       │
 │        ├── 有结果 ──→ 正常回复                  │
 │        └── 无结果 ──→ 回复"当前无法获取该信息"  │
 └───────────────────────────────────────────────┘

 ┌───────────────────────────────────────────────┐
 │  场景2: 模型 API 429(限频) / 超时              │
 │                                               │
 │  首选模型(如DS Flash) ──→ HTTP 429 / Timeout  │
 │        │                                       │
 │        ▼                                       │
 │  自动切换备选模型(如GLM-5.2) ──→ 重试请求       │
 │        │                                       │
 │        ├── 成功 ──→ 用备选模型回复（标注来源）  │
 │        └── 失败 ──→ 回复"服务暂时不可用"        │
 └───────────────────────────────────────────────┘

 ┌───────────────────────────────────────────────┐
 │  场景3: Gateway 进程宕机 / 端口不可达           │
 │                                               │
 │  Agent Gateway(:1300X) ──→ 连接失败            │
 │        │                                       │
 │        ▼                                       │
 │  飞书侧无响应 ──→ 用户需重启:                   │
 │  start-all-gateways.sh ──→ 10个Gateway并行恢复  │
 └───────────────────────────────────────────────┘

 ┌───────────────────────────────────────────────┐
 │  场景4: MCP Server 不可用                      │
 │                                               │
 │  MCP 调用超时 / 返回错误                        │
 │        │                                       │
 │        ▼                                       │
 │  重试3次(间隔2s)                                │
 │        │                                       │
 │        ├── 成功 ──→ 继续处理                    │
 │        └── 全部失败 ──→ 回复"该功能暂时无法使用" │
 └───────────────────────────────────────────────┘
```
